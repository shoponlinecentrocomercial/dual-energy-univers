# 🌌 Dual Energy Universe: Simulación de Red Cosmológica

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/shoponlinecentrocomercial/dual-energy-univers/HEAD?urlpath=%2Fdoc%2Ftree%2Fsimulacion.ipynb)

Simulación numérica del modelo cosmológico **"Universo de Energía Dual"**, donde el universo es una red de vórtices de energía que conecta materia y antimateria. El modelo predice la constante de Hubble (H₀ ≈ 70 km/s/Mpc) con un solo parámetro ajustable.

## 🚀 Ejecutar en Binder

Haz clic en el botón de arriba para lanzar el entorno interactivo en la nube (no necesita instalación).

## 📂 Contenido

- `simulacion.ipynb` - Notebook principal con la simulación.
- `sim_red_dual_completo.py` - Script en Python (versión standalone).
- `requirements.txt` - Dependencias para reproducir el entorno.

## 🔬 Resultados clave

- **H₀ = 69.98 km/s/Mpc** (observado: 70.0)
- La red de vórtices reproduce la expansión del universo sin materia ni energía oscura.
- La calibración del modelo usa dos parámetros: ξ (acoplamiento) y η (disipación topológica).
